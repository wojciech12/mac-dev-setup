# Linux / Ubuntu / Kubuntu

### Essentials

1. CLI

   ```bash
    sudo apt-get update && sudo apt-get install -y \
        gnupg \
        git \
        jq \
        golang-go \
        python3 \
        httpie \
        neovim
    ```

2. Fonts

   ```bash
   sudo apt-get install -y \
        fonts-firacode \
        fonts-jetbrains-mono
   ```
